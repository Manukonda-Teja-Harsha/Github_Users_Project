# 🌐 GitHub Users Explorer

A simple and responsive web application that fetches GitHub user information using the **GitHub Users API** and displays user details such as profile picture, username, and a button to visit the user's GitHub profile.

## 🚀 Features

- 🔍 Search for any GitHub user
- 🖼️ Display user profile image
- 👤 Show GitHub username
- 🔗 Visit the user's GitHub profile with a single click
- 📱 Responsive design for desktop and mobile devices
- ⚡ Fast data fetching using the GitHub REST API

## 🔗 GitHub Users API

This project uses the public GitHub REST API.

Example endpoint:

https://api.github.com/users/{username}

Example:

https://api.github.com/users/octocat

## 💻 How to Run

1. Clone the repository

```bash
git clone https://github.com/Manukonda-Teja-Harsha/Github_Users_Project.git
```

2. Navigate to the project folder

```bash
cd Github_Users_Project
```

3. Open `index.html` in your browser.

No additional setup is required.
